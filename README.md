# ESP32-IoT_based-Cable-Fault-Monitor
IoT-based underground cable fault distance locator using ESP32 and Blynk

Hardware required :
ESP32, Oled display, Relay, LM358, Potentiometers, Resistors

This project identifies the type of fault :
                                           1.Short circuit
                                           2.Open circuit
                                           3.Ground fault

Blynk setup : 
Virtual pins V0 - Status,String
             V1 - Distance,Double
             V2 - Reset,Integer
